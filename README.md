## 开发

```shell
git clone https://github.com/RealHeart/ZMusic -b mod
cd ZMusic
gradlew clean build
```

* [java-stream-player](https://github.com/goxr3plus/java-stream-player) 强大的Java音频库
* [AllMusic Mod](https://github.com/Coloryr/AllMusic_M) 模组通信相关思路
* [MCCustomSkinLoader](https://github.com/xfl03/MCCustomSkinLoader) 多加载器/MC版本集成相关思路
* [MinecraftForge](https://files.minecraftforge.net/) Minecraft Mod加载器
* [Fabric](https://fabricmc.net/) 新一代 Minecraft Mod加载器
* [JetBrains](https://www.jetbrains.com/zh-cn/) 提供强大的IDE支持
